# Humanoid Robot Fundamentals

This repository contains an educational Unitree G1 MuJoCo project and supporting humanoid robotics examples.

## Current task

The G1 moves its right arm forward and drops a 4 cm, 50 g ball into an 18 cm radius ground target directly in front of the robot. Finger control is out of scope; the scene uses a rigid five-finger visual hand.

The deterministic baseline reaches a 100% success rate across 10 seeded episodes, with a mean landing error of 0.081 m.

See the repository [README](../README.md) for setup, asset installation, baseline evaluation, and MuJoCo visualization commands.
